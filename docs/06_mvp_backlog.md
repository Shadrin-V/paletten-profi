# MVP Backlog

This document tracks the first implementation tasks and ERP workflows.

## MVP principle

The MVP should prove that the company can run core ERP operations in the new system with minimal customization.

## Not in first MVP

- Full marketplace integration
- Advanced AI layer
- Complex custom logistics optimization
- Deep automated supplier comparison
- Full custom ERP rewrite
